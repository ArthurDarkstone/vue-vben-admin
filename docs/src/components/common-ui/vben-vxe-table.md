# Vben Vxe Table 表格

框架提供的Table 列表组件基于 [vxe-table](https://xuliangzhan.github.io/vxe-table/)，结合`Vben Form表单`进行了二次封装。

其中，表头的 **表单搜索** 部分采用了`Vben Form表单`，表格主体部分使用了`vxe-grid`组件，支持表格的分页、排序、筛选等功能。
